Center Distance Designer — Legacy Migration

Stack: .NET 8, PHP, MySQL, ADO.NET, IIS/Linux

Reverse‑engineering and migrating a legacy engineering tool to a modern, maintainable stack.

Problem

The original .NET 4.0 app depended on deprecated servers and brittle UI flows, risking downtime and security issues.

Approach

Rebuilt dropdown/data flows, added PHP endpoints and MySQL procedures, and aligned deployment for Windows IIS and Linux (Cloudways).

Impact

Improved reliability, easier updates, and better security posture. Users get consistent dropdowns and faster page loads.
